# final-report-generation

## Purpose
Write the final report for a paper-agent run.

## Inputs
- All generated run artifacts
- Validation and review outputs

## Outputs
- `final_report.md`

## Procedure
1. Summarize the paper.
2. Summarize implementation targets and actual implementation.
3. Map paper components to code artifacts.
4. List used Skills and Tools.
5. Summarize evaluation results.
6. List differences, excluded items, and future work.
7. Document execution and setup.

## Rules
- The final report must be reproducible.
- Do not hide known limitations.
